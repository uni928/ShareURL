URL を便利機能付きで共有します。

共有された側は、アクセスするボタンで普通にアクセスできます。

共有された側がPCで共有された場合にスマホで見たいと思った場合、「URLをQRコードで表示するボタン」があり、「QRコード読み込みができるサイトの短縮URL」が記載されていますので、少ない手順でスマホで見れます。

また、「キャッシュ回避アクセス」ボタンを押すと、クエリパラメータに現在の時間を表す14文字が付与されるため、多くの場合キャッシュを回避して閲覧する事ができます。
